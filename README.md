# Amazon Clone 🛒

## 📌 Overview
This is a simple **Amazon Clone webpage** built using **HTML and CSS**.  
The project replicates the basic layout and design of Amazon’s homepage, focusing on responsive design and clean UI.

## 🎯 Features
- Responsive homepage layout  
- Navigation bar with categories  
- Product grid section  
- Footer with links and information  
- Styled using pure CSS (no frameworks)

## 🛠️ Technologies Used
- **HTML5** for structure  
- **CSS3** for styling and layout  

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/amazonclone.git
   ```
2. Navigate to the project folder:
   ```bash
   cd amazonclone
   ```
3. Open `index.html` in your browser.
